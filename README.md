# E-Commerce Platform

A full-stack e-commerce application built using React, Node.js, Express, MySQL, and JWT authentication.

## Features
- User authentication (JWT-based)
- Product listing and management
- Shopping cart functionality
- Secure backend APIs
- Responsive frontend UI

## Tech Stack
- Frontend: React, HTML, CSS
- Backend: Node.js, Express
- Database: MySQL
- Authentication: JWT

## How to Run

1. Clone the repository  
   `git clone https://github.com/harshpandey9198/ecommerce-platform.git`

2. Install backend dependencies  
   `npm install`

3. Start backend server  
   `node server.js`

4. Open frontend  
   Open `index.html` in your browser
